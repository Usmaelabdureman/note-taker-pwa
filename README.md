# Note Taker PWA

This is a **Progressive Web Application (PWA)** that allows users to take and save notes directly in their browser. The app is built using **HTML**, **CSS**, and **JavaScript** and utilizes the browser's local storage to persist user notes, making it available offline after the initial load.

## Features

- Create, view, and delete notes
- Offline functionality using PWA capabilities
- Simple and intuitive user interface
- Responsive design for mobile and desktop views
- Local storage for saving notes

## Installation

To run this app locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/note-taker-pwa.git
   ```
2. Navigate to the project directory:
   ```bash
   cd note-taker-pwa
   ```
3. Open `index.html` in your browser:
   ```bash
   open index.html
   ```

## Usage

- Open the app in your browser.
- You can add a new note using the input field and clicking on the "Add" button.
- The notes will be saved to your local storage and will persist even when the app is used offline.
- To delete a note, simply click the "Delete" button next to the corresponding note.

## Progressive Web App (PWA)

This app can be installed on your device for a native-like experience. Follow these steps:

1. Open the app in a modern web browser like Chrome or Edge.
2. Click on the install icon in the URL bar (usually a "+" icon).
3. Follow the on-screen prompts to add it to your home screen or desktop.
4. The app will now be available as a standalone application on your device.
